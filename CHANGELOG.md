# Changelog

All notable changes to ELIA OS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-01-15

### Added
- **Solana Wallet Integration** - Multi-wallet support (Phantom, Backpack, Solflare)
- **Unique Agent ID System** - Browser fingerprinting and disqualification protection
- **ASCII Intro Animation** - Dramatic letter-by-letter reveal on page load
- **Background Audio** - Auto-playing background music (elia.mp3)
- **Express.js Backend** - Production-ready REST API server
- **Operations Room** - Multi-target surveillance with 4 active targets
- **Intelligence Marketplace** - 6 intelligence gathering tools
- **Encrypted Vault** - 9 files with interactive decrypt functionality
- **Secure Terminal** - 15+ working commands with real outputs
- **API Endpoints** - Wallet registration, intelligence extraction, system status
- **GitHub Ready** - Complete documentation, package.json, server.js

### Changed
- Transformed from MONOKURO to ELIA OS theme
- Updated all text from personal narrative to corporate espionage
- Redesigned all icons to glitchy gray aesthetic (◆ ▓ ◉)
- Improved responsive design for all screen sizes
- Enhanced terminal with real-time state tracking

### Fixed
- Wallet detection timing issues (100ms retry logic)
- Responsive layout at 100% zoom
- Terminal command execution
- File decryption animations
- Search functionality in marketplace
